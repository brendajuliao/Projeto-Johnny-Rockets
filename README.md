# Projeto Johnny Rockets Web

Este projeto consiste no desenvolvimento de um site inspirado no estilo da rede de restaurantes Johnny Rockets, criado durante o 2º semestre da faculdade.

A aplicação simula o ambiente de uma hamburgueria, incluindo páginas informativas e integração com back-end em PHP, permitindo a manipulação e exibição de dados.

## Tecnologias utilizadas
- HTML
- CSS
- PHP
- MySQL

## Funcionalidades
- Página inicial com identidade visual da marca
- Página de promoções
- Página de lojas
- Página de delivery
- Página de contato
- Integração com banco de dados
- Listagem de dados utilizando o arquivo `listar_usuarios.php`

## Back-end e Banco de Dados
O projeto possui integração com banco de dados através do arquivo `config.php`, responsável pela conexão.

Os dados podem ser manipulados e exibidos dinamicamente utilizando scripts em PHP.

## Objetivo do projeto
Aplicar na prática conceitos de:
- Desenvolvimento web front-end e back-end
- Integração com banco de dados
- Estruturação de aplicações web
- Organização de páginas e navegação
