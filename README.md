#  Projeto Johnny Rockets Web

Este projeto consiste no desenvolvimento de um site inspirado no estilo da rede de restaurantes Johnny Rockets, criado durante o 2º semestre da faculdade.

O objetivo foi simular a interface de uma hamburgueria moderna, com foco na apresentação visual dos produtos, navegação intuitiva e organização das informações para o usuário.

## Tecnologias utilizadas
- HTML
- CSS

## Funcionalidades
- Página inicial com destaque para a identidade visual da marca
- Seção de produtos (hambúrgueres e itens do cardápio)
- Página de contato
- Estrutura de navegação entre páginas

## Objetivo do projeto
Aplicar na prática os conceitos de desenvolvimento web aprendidos em sala de aula, como:
- Estruturação de páginas com HTML
- Estilização com CSS
- Organização de layout e design responsivo básico

## Status
Projeto concluído para fins acadêmicos
